# Yummy_App
React Project :
<ul>
  <li>Showing the food with its recipes</li>
  <li>Showing food in many countries</li>
  <li>This project made by single page application (React Router Dom)</li>
  <li>Uses many dependances such as react-detect-offline , Axios , BootStrap and fontawesome</li>
  <li>Web API with (Meal DB API)</li>
